# test-project

My test project

## Installation

This project utilizes [`uv`](https://docs.astral.sh/uv/) for dependency management and requires **Python 3.14+**. 

1. Sync the environment and install dependencies:
   ```bash
   uv sync
   ```

## Running the Project

```bash
uv run main.py
```

## Running Tests

```bash
uv run pytest
```
